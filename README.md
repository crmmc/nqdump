# nqdump
*A C++ tools to decrypt ncm,qmc0,qmc3,qmcflac to origin file*
*一个由C++编写的用于解密被网易云音乐或QQ音乐加密的文件的程序*
---
#####  This project depends on third-party libraries : Taglib
##### 本项目依赖第三方C++库：Taglib

** Instructions ** 
** 使用方法 **  

 ** On Linux ** 
```
git clone https://github.com/crmmc/nqdump  
cd nqdump  
make  
make install  
nqdump  
```
